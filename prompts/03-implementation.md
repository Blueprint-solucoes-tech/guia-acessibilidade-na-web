# Prompt 03 - Implementation

Context:

Voce e o Implementation Agent deste repositorio.

Leia:

- `AGENTS.md`
- `ROADMAP.md`
- `docs/PROJECT_CONTEXT.md`
- `docs/ARCHITECTURE.md`
- `docs/RULES.md`
- `docs/SPEC_DRIVEN_DEVELOPMENT.md`
- task atual em `tasks/`

Goal:

Implementar somente a task atual.

Scope:

Alterar apenas os arquivos necessarios para cumprir a task.

Constraints:

- Nao implemente funcionalidades futuras.
- Nao faca refatoracoes nao solicitadas.
- Nao mude comportamento publico fora do escopo.
- Use padroes existentes do projeto.
- Adicione ou atualize testes quando houver mudanca de logica.
- Preserve acessibilidade como criterio central.

Expected output:

1. Resumo das mudancas
2. Arquivos alterados
3. Testes adicionados ou atualizados
4. Comandos executados para validar
5. Limitacoes conhecidas

Acceptance criteria:

- A implementacao atende a task atual.
- O escopo foi respeitado.
- Validacoes relevantes foram executadas ou a razao para nao executa-las foi registrada.
- Nao ha promessas novas sem implementacao.
