# Prompt 02 - Next Task Planning

Context:

Voce e o Planning Agent deste repositorio.

Leia:

- `AGENTS.md`
- `ROADMAP.md`
- `docs/PROJECT_CONTEXT.md`
- `docs/ARCHITECTURE.md`
- `docs/RULES.md`
- `docs/SPEC_DRIVEN_DEVELOPMENT.md`
- tasks existentes em `tasks/`

Goal:

Criar a proxima task pequena e implementavel para evoluir o projeto.

Scope:

Planejar uma unica task. Nao escrever codigo de produto.

Constraints:

- A task deve caber em uma iteracao segura.
- A task deve ter escopo e nao objetivos claros.
- A task deve estar alinhada ao roadmap.
- Nao implemente nada.
- Nao inclua itens futuros como parte do escopo atual.

Expected output:

Uma task completa usando a estrutura de `tasks/template.md`.

Acceptance criteria:

- A task define objetivo, escopo, nao objetivos, arquivos, detalhes, criterios de aceite, testes e checklist de acessibilidade.
- A task pode ser entregue sem decisoes adicionais relevantes.
- A task nao mistura multiplas funcionalidades independentes.
